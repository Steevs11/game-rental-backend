# 🎮 Game Rental Backend

Backend aplikacija za iznajmljivanje igrica napravljena u Node.js koristeći Express.

## 🚀 Funkcionalnosti

- CRUD operacije za igrice
- Kreiranje i vraćanje iznajmljivanja
- Automatsko računanje cene
- Dodatni trošak za kašnjenje
- Ocenjivanje igrica
- Top 3 najbolje ocenjene igrice
- Filtriranje igrica po žanru, platformi i oceni
- Prikaz dostupnih i iznajmljenih igrica
- Istorija iznajmljivanja
- Pregled iznajmljivanja po korisniku

## 🛠 Tehnologije

- Node.js
- Express
- JSON (file-based storage)

## ▶️ Pokretanje projekta

npm install  
npm run dev  

Aplikacija se pokreće na:

http://localhost:3000

## 🧪 Testiranje

API se testira pomoću Postman-a.

## 📂 Struktura projekta

controllers/ – logika aplikacije  
routes/ – API rute  
utils/ – pomoćne funkcije  
data/ – JSON podaci
